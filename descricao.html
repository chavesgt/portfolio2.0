Descrição📑

Sou técnico de informática e suporte de TI há mais de 14 anos, agora estou migrando para a área de desenvolvimento de software com o objetivo de se tornar programador.

Iniciei na área de tecnologia aos 16 anos por curiosade e me tornei técnico de informática pouco tempo depois trabalhando como micro empreendor executando serviços de informática em geral. Trabalhei por mais de 6 anos prestando suporte técnico para clientes e funcionários do Banco Do Brasil (CLT).

Agora estou focado em entrar na área da programação iniciei a graduação de análise e desenvolvimento de sistemas e também o curso técnico de programador full-stack em paralelo.

Tenho alguns projetos desenvolvidos em meu GitHub com o uso das tecnologias Front-End ( HTML5 - CSS 3 - Bootstrap - Angular - JavaScript - TypeScript), Fremework Front-End com consumo de API e Back-End ( MySql - PHP ).

Outros conhecimentos em tecnologias: Postman - Selenium - PHPMyAdmin - Jasmine - Jmeter - UML - UI | UX Designer - XAMPP | WampServer - Git | GitHub - Linux | Bash - Cloud Computing (Google Cloud | AWS | Azure) .NET - Python - C# - C++ - Java - Json - Node.JS .

Conhecimentos de Suporte TI: Windows Avançado - PowerShell - Hardware - Software - Active Directory - Linux - MacOS - Redes - Impressoras - Suporte Remoto - Android - Pacote Office - Segurança Cibernética - Backup e Restauração.
